Key Features

  • Frame Selected object or vertex/edge/face selection.
  • Frame all if nothing is selected.
  • Works in both Object Mode and Edit Mode
  • Automatically zoom out the 3D View to a reasonable distance when only 1 vertex is selected, without disturbing orbit.

Installation

  1. Download the Neltulz - Smart Frame Selection zip file anywhere on your computer.
  2. Run Blender.
  3. Go to Edit > Preferences.
  4. In the left sidebar, go to Add-ons.
  5. At the top right, click Install.
  6. Browse for the Neltulz - Smart Frame Selection zip file and click Install Add-on from File
    Alternatively, you may double click the zip file to install it.
  7. Next to Generic: Neltulz - Smart Frame Selection, enable the checkbox.
  8. At the lower left of the preferences window, click Save Preferences and close the preferences window.

How to Use

Hotkeys
Note: Hotkey works in both "Object" and "Mesh" mode
Hotkey Result Command Properties
F Frame object.neltulz_smart_frame_sel frameSelection = True
isolateSelection = False
SHIFT+F Isolate object.neltulz_smart_frame_sel frameSelection = False
isolateSelection = True
CTRL+SHIFT+F Frame & Isolate object.neltulz_smart_frame_sel frameSelection = True
isolateSelection = True

Known Issues & Limitations

  • If you isolate while in object mode, then try to isolate again while in edit mode, you'll have to perform the "isolate" operator twice to get it to work.

Changelog

  • 2019-12-07
    • Added ability to hide axes when isolating objects
    • Added "Use Extreme Isolate". When enabled, hidden objects will be "truly" hidden, granting a huge performance increase when adding objects to the scene and adjusting their operator properties.

      Use caution, this setting can cause the scene to take a very long time to refresh when unhiding all previously hidden objects, depending on scene complexity and total scene poly count.
    • Improve compatibility with other addons when performing UNDO or REDO. (Added "UNDO" to the operator options
    • Removed some console print messages.
    • Removed some unnecessary code that apparently did nothing.
    • Removed some code that was supposed to improve camera distance when framing only 1 vertice, but in many cases, was causing the camera to be too zoomed out.
  • 2019-10-12b
    • Added ability to move the viewport to the origin
    • More object type support for framing and isolating. (Surface, Metaball, Grease Pencil, Armatures, etc.)
    • Organized & improved the sidebar panel
    • Added ability to use all regions when framing (Useful for quad view)
    • Simplified what is framed when nothing is selected
    • More Hotkey (Keymaps)
  • 2019-10-12
    • Added ability to "Hide Floor on Isolate" (Enabled by default)
    • Added ability to "Frame" and "Isolate" Bezier Curve points
    • Added Error messages when trying to "Frame" or "Isolate" unsupported object types while in "Edit" mode
    • Added "Use Advanced Settings" checkbox and "Show Error Messages" checkbox
  • 2019-10-09
    • Added ability to collapse/expand list of excluded objects from isolate to prevent the panel from skyrocketing in height
    • Added ability to turn objects into templates (wireframe display with select-through capability).
  • 2019-10-05 - Added ability to exclude certain objects from isolate select. Useful for excluding image planes. Suggested by Piotr Jedzinski
  • 2019-08-28 - Fix bug when trying to frame all when there are no mesh objects selected in the scene.
  • 2019-08-20
    • Added isolate support
    • Framing while nothing is selected now frames only selected mesh objects by default
    • Configure exactly what is framed when nothing is selected.
    • Sidebar Pane with buttons
    • 2 new hotkeys (SHIFT+F to Isolate) (CTRL+SHIFT+F to Frame & Isolate)
  • 2019-08-19 - Fixed panel showing things from another addon. Disabled panel until needed.
  • 2019-08-18 - Changed license from "CC BY 4.0" to GNU GPL v3.0 because it's easier to select on Github
  • 2019-08-17 - First release!
