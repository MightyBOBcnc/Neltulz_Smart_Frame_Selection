Key Features

  • Frame Selected object or vertex/edge/face selection.
  • Frame all if nothing is selected.
  • Works in both Object Mode and Edit Mode
  • Automatically zoom out the 3D View to a reasonable distance when only 1 vertex is selected, without disturbing orbit.

Installation

  1. Download the Neltulz - Smart Frame zip file anywhere on your computer.
  2. Run Blender.
  3. Go to Edit > Preferences.
  4. In the left sidebar, go to Add-ons.
  5. At the top right, click Install.
  6. Browse for the Neltulz - Smart Frame zip file and click Install Add-on from File
    Alternatively, you may double click the zip file to install it.
  7. Next to Generic: Neltulz - Smart Frame, enable the checkbox.
  8. At the lower left of the preferences window, click Save Preferences and close the preferences window.

How to Use

Hotkeys
Note: Hotkey works in both "Object" and "Mesh" mode
Hotkey Result Command Properties
F Frame view3d.ntzsf_smart_frame frameSelection = True
isolateSelection = False
SHIFT+F Isolate view3d.ntzsf_smart_frame frameSelection = False
isolateSelection = True
CTRL+SHIFT+F Frame & Isolate view3d.ntzsf_smart_frame frameSelection = True
isolateSelection = True

Known Issues & Limitations

  • If you isolate while in object mode, then try to isolate again while in edit mode, you'll have to perform the "isolate" operator twice to get it to work.
